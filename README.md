# zayer_app

A Flutter logistics shopping app (Zayer).

## Platform Notes

**WebView is supported on Android and iOS only.** When running on Windows, Linux, or Web:
- Tapping "Shop on Amazon" (or any store) opens a fallback screen instead of crashing.
- The fallback shows: "WebView is supported on Android/iOS only. Run on an emulator/device." plus the store URL and an "Open in browser" button.

To test the full WebView flow (in-app store browsing, PDP detection, Add to Cart overlay), run the app on an **Android emulator** or **iOS device**.
